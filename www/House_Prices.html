1. Introduction

Objective: To analyze and model the housing dataset to predict factors like price based on various attributes such as area, number of bedrooms, and locality.

Dataset Overview: The dataset contains housing details, including size, price, location, and other features that can influence housing prices.

2. Dataset Description

Summary: The dataset includes the following features:

  • area: Size of the house in square feet.
  • bedrooms: Number of bedrooms in the house.
  • price: Price of the house.
  • bathrooms: Number of bathrooms in the house.
  • stories: Number of stories.
  • mainroad: Whether the house is near the main road (yes/no).
  • guestroom: Presence of a guest room (yes/no).
  • basement: Presence of a basement (yes/no).
  • hotwaterheating: Availability of hot water heating (yes/no).
  • airconditioning: Presence of air conditioning (yes/no).
  • parking: Number of parking spaces.
  • prefarea: Whether the house is in a preferred area (yes/no).
  • furnishingstatus: Furnishing status of the house (furnished/semi-furnished/unfurnished).

3. Preprocessing

Steps Taken: Applied the following preprocessing steps:

  • the dataset does not contain any missing values, no need to handle them.
  • Converted all categorical variables like mainroad and guestroom to numerical using label encoding.
  • Normalized numerical features such as bedrooms,bathrooms,stories and parking using Min-Max scaling.

Rationale: Preprocessing ensures data consistency, handles missing values, and prepares the data for machine learning models.

4. Analysis

Univariate Analysis

Generated histograms for numerical features like price and area, and pie charts for categorical variables like furnishingstatus.

Bivariate Analysis

Scatter plots reveal strong positive correlations between area and price. Bar plots highlight the impact of preferred area on pricing.

5. Modeling

Target Variable: price, a continuous variable.

Selected Models: Linear Regression and Decision Tree were chosen to predict housing prices.

Evaluation Metrics: Metrics include Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and R-squared.

Data splitting: we have taken 70% of the data as training and 30% as testing
