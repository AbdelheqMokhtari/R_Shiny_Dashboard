1. Introduction

Objective: To analyze and model the diabetes diagnosis dataset to predict whether a person has diabetes based on clinical and personal attributes.

Dataset Overview: The dataset contains information about patients, including age, gender, BMI, blood pressure, glucose levels, and whether they were diagnosed with diabetes.

2. Dataset Description

Summary: The dataset includes the following features:

  • patientId: Unique identifier for each patient.
  • age: Age of the patient (numerical).
  • gender: Gender of the patient (0 = female, 1 = male).
  • BMI: Body Mass Index (numerical).
  • blood_pressure: Blood pressure levels (numerical).
  • insuline: Insulin levels (numerical).
  • glucose: Glucose levels (numerical).
  • diabetesPedigreeFunction: Genetic predisposition to diabetes (numerical).
  • outcome: Target variable (0 = no diabetes, 1 = diabetes).

Exploratory Insights: Initial exploration reveals a strong correlation between high glucose levels, BMI, and the likelihood of diabetes. Patients with higher diabetes pedigree function values also show a higher chance of being diagnosed.

3. Preprocessing

Steps Taken: Applied the following preprocessing steps:

  • Dropping the feauture PatientID because it only represents the ID of the patient which is not pertinent feauture to train.
  • In this dataset does not contain any missing values
  • The feauture gender and outcome are already encoded.
  • Normalized numerical features like BMI, glucose, and insuline using log transformation.

Rationale: Preprocessing ensures the dataset is clean and ready for modeling, with consistent formats for numerical and categorical variables.

4. Analysis

Univariate Analysis

Generated histograms for numerical features like glucose, BMI, and age, and bar plots for categorical variables like gender.

Bivariate Analysis

The correlation matrix reveals key relationships among the numerical features in the dataset:

  • glucose and outcome show a strong positive correlation, indicating that higher glucose levels are strongly associated with diabetes diagnosis.
  • BMI and Insulin also has a moderate positive correlation with outcome, suggesting that higher BMI and Insulin increases the likelihood of diabetes.
  • Features like gender and blood_pressure exhibit weaker correlations with outcome, but they still provide some predictive value.

A heatmap of the correlation matrix highlights these relationships, with darker shades indicating stronger correlations. This analysis helps identify the most influential predictors for the diabetes diagnosis model.

5. Modeling

Target Variable: outcome, a binary categorical variable indicating the presence or absence of diabetes.

Selected Models: svm and Random Forest were chosen for classification.

Evaluation Metrics: Metrics include accuracy, precision, recall,specitivity and F1 score, and the area under the ROC curve (AUC).
